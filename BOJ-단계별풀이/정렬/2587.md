##  풀이

문제에 입력받는 수는 5개로 정해져 있다. 입력받을 때마다 선언한 배열에 추가해 준다. </br>
평균은 `reduce`를 고차함수를 사용하여 평균을 구하고, `sorted`를 사용하여 중앙값을 구한다.

```Swift
var nums: [Int] = []

for _ in 0..<5 {
    nums.append(Int(readLine()!)!)
}

print(nums.reduce(0, +) / 5)
print(nums.sorted(by: <)[2])

```
