##  풀이

내장된 정렬 함수를 사용하여 풀이했다.. `sorted()`로 정렬한 후 `forEach`로 줄바꿈하여 출력했다.

```Swift
let N: Int = Int(readLine()!)!
var nums: [Int] = []

for _ in 0..<N {
    nums.append(Int(readLine()!)!)
}

nums.sorted().forEach {
    print($0)
}
```
