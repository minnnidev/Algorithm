##  풀이

`String`으로 숫자들을 배열로 받고 순회하면서 합을 구해준다. 순회할 때 각 숫자들의 타입은 `Character`임으로 `String`으로 변환 후 `Int`로 변환 필요함.


## 코드 

1) for 문 사용

```Swift
let num: Int = Int(readLine()!)!
let nums: String =  readLine()!
var sum: Int = 0

for num in nums {
    sum += Int(String(num))!
}

print(sum)
```

</br>

2) `map`, `reduce`

```Swift
let num: Int = Int(readLine()!)!
let nums: String =  readLine()!

print(nums.map { Int(String($0))! }.reduce(0, +))
```
