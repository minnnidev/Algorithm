##  풀이 

`joined()`를 사용하기 위해 String 배열로 입력을 받을 수 있다고 판단했다. 내림차순으로 정렬한 뒤 `joined()`를 사용하여 요소들을 합쳐 출력하였다.

```Swift
let num: [String] = readLine()!.map { String($0) }
print(num.sorted(by: >).joined())
```
