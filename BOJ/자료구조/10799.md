##  풀이

stack: 막대기 수 계산을 위한 stack </br>
answer: 총 결단된 막대기 수

```
( -> 막대기이므로 stack에 +1 
) -> 1. 앞에 ( 이면 레이저이므로 stack - 1 하기. (레이저의 ( 이므로) 
        answer에 지금까지 계싼된 막대기 수(stack) 더해 주기 
     2. 앞이 ) 이면 막대기의 끝이므로 answer + 1, stack - 1 (막대가 사라지므로)  
```

```Swift
let line: [String] = readLine()!.map { String($0) }
var stack: Int = 0
var answer: Int = 0

for i in 0..<line.count {
    if line[i] == "(" {
        stack += 1
    } else {
        stack -= 1
        if line[i-1] == ")" { // 막대기의 끝
            answer += 1
        } else { // 레이저
            answer += stack
        }
    }
}

print(answer)
```
