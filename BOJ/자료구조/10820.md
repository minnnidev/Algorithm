##  풀이

쉬운 문제이지만 특이한 점: 테스트 케이스 수를 따로 입력받지 않음 </br>
입력을 받을 때에 케이스를 나눠 각 변수에 +1 해 준다. </br>
대소문자는 아스키코드로 구분해 준다. </br>

```Swift
while let line = readLine() {
    var lower: Int = 0
    var upper: Int = 0
    var num: Int = 0
    var space: Int = 0
    
    for char in line {
        if "a" <= char && char <= "z" {
            lower += 1
        } else if "A" <= char && char <= "Z" {
            upper += 1
        } else if char == " " {
            space += 1
        } else {
            num += 1
        }
    }
    print(lower, upper, num, space)
}
```
