##  풀이

덱(deque): 양쪽 끝에서 삽입과 삭제를 할 수 있는 자료구조
```
push_front X: X를 맨 앞에 추가
push_back X: X를 맨 뒤에 추가
pop_front: 빈 배열이 아닐 때 맨 앞 요소를 삭제 
pop_back: 빈 배열이 아닐 때 맨 뒤 요소를 삭제 
size: 배열의 요소 개수 출력
empty: 빈 배열인지 체크
front: 빈 배열이 아닐 때 맨 앞 요소 출력
back: 빈 배열이 아닐 때 맨 뒤 요소 출력
```

1. 

```Swift
let N: Int = Int(readLine()!)!
var deque: [Int] = []

for _ in 0..<N {
    let command: [String] = readLine()!.split(separator: " ").map { String($0) }

    switch command[0] {
    case "push_front":
        deque.insert(Int(command[1])!, at: 0)
    case "push_back":
        deque.append(Int(command[1])!)
    case "pop_front":
        print(deque.isEmpty ? "-1" : deque.removeFirst())
    case "pop_back":
        print(deque.isEmpty ? "-1" : deque.removeLast())
    case "size":
        print(deque.count)
    case "empty":
        print(deque.isEmpty ? "1" : "0")
    case "front":
        print(deque.isEmpty ? "-1" : deque.first!)
    case "back":
        print(deque.isEmpty ? "-1" : deque.last!)
    default:
        print("error")
    }
}
```
